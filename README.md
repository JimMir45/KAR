# KAR 赛车俱乐部小程序

KAR赛车俱乐部微信小程序，为俱乐部提供线上化运营工具。

## 功能概览

- **赛道日活动管理** — 发布活动、在线报名、微信支付、增值服务
- **商城** — 轮胎、刹车、机油等实物商品销售，到店安装
- **圈速排行榜** — 按赛道分类，支持车型搜索
- **会员体系** — 滚动12月消费升级，储值充值，等级折扣
- **财务管理** — 收入自动归集，支出录入，固定资产，月度报表
- **库存管理** — 配件出入库，库存预警
- **数据看板** — 经营概览，周会看板
- **供应商管理** — 采购订单，对账

## 技术栈

| 层 | 技术 |
|---|------|
| 后端 | Go + Gin + GORM + Casbin + PostgreSQL |
| 前端 | Taro 4.x + React + TypeScript + NutUI React |
| 微信SDK | silenceper/wechat + wechatpay-go |
| 部署 | Docker Compose + Caddy + 腾讯云 |
| 存储 | PostgreSQL + 腾讯云COS |
