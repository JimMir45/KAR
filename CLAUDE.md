# KAR 赛车俱乐部小程序

## 项目简介

KAR赛车俱乐部微信小程序，覆盖赛道日活动管理、商城、圈速排行、会员运营、财务管理等核心业务。C端（用户端）+ B端（管理端）双小程序架构。

## 技术栈

- **后端**: Go + Gin + GORM + Casbin + PostgreSQL
- **前端**: Taro 4.x + React + TypeScript + NutUI React
- **微信SDK**: silenceper/wechat（登录/消息）+ wechatpay-go（支付）
- **部署**: Docker Compose + Caddy（自动HTTPS）+ 腾讯云轻量服务器
- **存储**: 腾讯云COS（图片）+ PostgreSQL（数据）
- **Monorepo**: pnpm Workspaces

## 开发规范

- 后端项目结构遵循 Clean Architecture：handler → service → repository → domain
- 所有金额字段使用整数（单位：分），避免浮点数精度问题
- API统一响应格式：`{ code, message, data }`
- Git commit 遵循 Conventional Commits：`feat:` / `fix:` / `docs:` / `refactor:`
- 数据库迁移使用 Atlas，开发阶段可用 GORM AutoMigrate
- 前端组件文件 < 400行，后端函数 < 50行
